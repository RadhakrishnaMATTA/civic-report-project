## 🌍 Project Overview

The **Civic Report System** is a web application that allows citizens to report local issues such as:

* 🛣️ Road damage
* 💡 Street light problems
* 🚮 Garbage issues
* 🚰 Water leakage

📢 This system helps **bridge the gap between citizens and authorities** by enabling quick issue reporting and tracking.

---

## 🎯 Key Features

✔️ Citizens can submit complaints easily
✔️ Categorized issue reporting (roads, water, electricity, etc.)
✔️ Real-time data handling
✔️ Admin/Manager can view and manage reports
✔️ Clean and user-friendly interface
✔️ REST API integration

## ⚙️ Setup Instructions

### 🔹 1. Clone Repository

```bash id="f5mubn"
git clone https://github.com/your-username/civic-report-system.git
cd civic-report-system
```

---

### 🔹 2. Backend Setup (Spring Boot)

```bash id="pajq4d"
cd backend
mvn spring-boot:run
```

---

### 🔹 3. Frontend Setup (React)

```bash id="p7iv7j"
cd frontend
npm install
npm start
```
